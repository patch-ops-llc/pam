Detailed Scope of Work

Story ID Hours Workstream Name Customer Story Recommended Approach Assumptions
1 10 FedEx API Integration Setup Team needs secure, authenticated connection to FedEx Rate API to retrieve real-time shipping costs based on package specifications → Set up FedEx developer account and obtain API credentials
→ Configure OAuth 2.0 authentication for FedEx Rate API v1
→ Test API connection and validate rate retrieval functionality 		→ Requires Data Hub Professional subscription for serverless functions
→ FedEx developer account accessible or can be created during implementation
2 10 Deal Property Configuration Sales team needs structured fields to capture shipping parameters that feed into the FedEx API call → Create custom deal properties for shipping weight (numeric, pounds)
→ Create custom deal property for destination ZIP code (text, 5-digit validation)
→ Configure origin ZIP code property (default to warehouse location)
→ Set up calculated shipping cost property to store FedEx response
→ Add shipping service level property (Ground, Express, Overnight options) 		→ Standard package dimensions can be used (no custom sizing required)
3 35 Custom UI Extension Development Sales team needs intuitive interface on deal records to input shipping details, retrieve FedEx rates, and see results without leaving the deal view → Build custom UI card extension using HubSpot CRM development tools
→ Design input form with fields for weight (numeric) and destination ZIP code
→ Implement client-side validation for required fields and data formats
→ Build API call function to FedEx Rate API with proper error handling
→ Create response parser to extract shipping cost from FedEx JSON response
→ Design result display showing shipping cost with service level breakdown
→ Style UI card with professional design matching HubSpot interface patterns
→ Implement loading states and user feedback during API calls 		→ UI extension deployed to deal record middle column
→ Requires Content Hub Enterprise subscription for custom code
→ Real-time API calls acceptable (no batch processing needed)
→ Standard HubSpot UI components and styling sufficient
4 20 Line Item Integration Logic Sales team needs shipping costs automatically written to deal line items for accurate quote generation without manual entry → Build line item creation function to add "Shipping" product to deal
→ Configure logic to check for existing shipping line items before creating duplicates
→ Implement update function to modify existing shipping line item price
→ Create or configure "Shipping" product in HubSpot product library
→ Set up line item properties to store FedEx service level and quote timestamp
→ Build recalculate button that updates existing line item with fresh FedEx rate
→ Implement deal total recalculation after line item changes 		→ "Shipping" configured as non-inventory product in HubSpot
→ Only one shipping line item per deal (single shipment model)
→ Recalculation overwrites previous shipping cost without approval workflow
